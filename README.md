DExploratory Data Analysis Project Welcome to my ** Exploratory Data Analysis Project ** repository . This project demonstrates how we use data available in a data warehouse for EDA before advanced analytics in order to derive actionable insights for a data scientist,business analyst and various other teams. It includes all the best practices and processes that a standard industry follows.

Project Requirements Objective Perform various advanced Data Analytical Methods on the data available in the gold layer of the Data Warehouse created using Medallion Architecture.
